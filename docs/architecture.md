# Архитектура CAIRN

> Подробное описание – в документе «Описание архитектуры CAIRN» [архитектура.docx](архитектура.docx).
> Данный файл – краткий технический справочник для разработчика.

## Три фазы обработки

```
Данные мониторинга → [Восприятие] → [Рассуждение] → [Объяснение] → Отчёт
```

## Обучение (`src/cairn/training/`)

### Трёхэтапная стратегия

1. **50 эпох** – нормальные данные: L_УМ + L_ВАК
2. **100 эпох** – аномальные данные: L_ПЭ + L_нез + L_КР + L_реб (первые 10 эпох – заморозка)
3. **30 эпох** – совместная оптимизация всех компонентов

### Функция потерь (формула 5.1)

```
L = λ₁·L_ПЭ + λ₂·L_УМ + λ₃·L_ВАК + λ₄·L_нез + λ₅·L_КР + λ₆·L_реб
```
